---
title: 'List Sources'
api: 'GET /api/v1/sources'
description: 'Retrieve all document sources for your organization'
---

## Request

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token containing your API key
</ParamField>

## Response

Returns an array of source objects.

<ResponseField name="id" type="string">
  The unique identifier of the source
</ResponseField>

<ResponseField name="fileName" type="string">
  The name of the uploaded file
</ResponseField>

<ResponseField name="fileType" type="string">
  The MIME type of the uploaded file
</ResponseField>

<ResponseField name="nickName" type="string">
  A user-friendly name for the source
</ResponseField>

<ResponseField name="isIndexed" type="boolean">
  Whether the source has been fully indexed
</ResponseField>

<ResponseField name="isIndexing" type="boolean">
  Whether the source is currently being indexed
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO timestamp of when the source was created
</ResponseField>

<ResponseField name="updatedAt" type="string">
  ISO timestamp of when the source was last updated
</ResponseField>

<RequestExample>

bash cURL
curl -X GET https://cloud.rowfill.com/api/v1/sources \
-H "Authorization: Bearer {api_key}"

<ResponseExample>
json Response
[
{
"id": "clg2xyzabc123",
"fileName": "document.pdf",
"fileType": "application/pdf",
"nickName": "document.pdf",
"isIndexed": true,
"isIndexing": false,
"createdAt": "2024-03-15T12:00:00Z",
"updatedAt": "2024-03-15T12:05:00Z"
}
]

</ResponseExample>
